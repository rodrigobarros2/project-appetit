# Documentação
    
## Como rodar o projeto localmente.

```Clonando o repositório

git clone https://github.com/rodrigobarros2/project-appetit.git<br />

Escolha o diretorio

cd project-appetit

Instalando dependencias

npm install
ou
yarn install

Iniciar aplicação

npm start
ou
yarn start
```
