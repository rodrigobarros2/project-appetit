How do I run the demos locally?
Clone this repo<br />
git clone https://github.com/rodrigobarros2/project-appetit.git<br />
Change directory<br />
cd project-appetit<br />
Install dependencies<br />
npm install<br />
or<br />

yarn install<br />
Run the app<br />
npm dev<br />
or<br />

yarn dev<br />
